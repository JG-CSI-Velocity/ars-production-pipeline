Format ODD Files
Extract and format raw ODD files from the CSM data dump folder. This prepares the data for analysis.
Select What to Format
Choose your name, the period, and optionally a specific client. Leave client blank to format all.

Period
Client ID (blank = all)
What This Does
1
Extract
Copies ZIP from your CSM data dump folder. Extracts the ODD CSV. Never modifies the original.
2
Format
Runs the 7-step formatting: totals, averages, swipe categories, ages, response grouping, segmentation.
3
Save
Writes the formatted Excel to 02-Data-Ready for Analysis, verified and ready for the Generate step.
Don't have access to a CSM's raw dump folder? Drop pre-formatted ODD files here ›
Stage pre-formatted ODD files
Skips the 7-step formatter. Use when someone hands you an already-extracted ODD .xlsx for a CSM whose raw dump folder you can't see (Dan, Aaron, etc.).
